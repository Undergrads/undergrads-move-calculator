# Undergrads Move Calculator

A simple web-based calculator to help customers estimate:
- How many movers they need
- How many hours the move will take

Based on:
- Residence type (studio, 1BR, 2BR, house, etc.)
- Number of rooms
- Furniture inventory
- Move distance/location
- Type of move (Load, Unload, Load & Unload, In-house)
